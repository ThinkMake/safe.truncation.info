definition: |
  Subset is a set whose elements are all contained within a larger set. From sub- plus set. A fundamental concept in mathematics and logic: if every member of set A is also a member of set B, then A is a subset of B. Used broadly in data analysis, computing, and everyday speech to describe a smaller portion within a larger group. Neutral&nbsp;register.
