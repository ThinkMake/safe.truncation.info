  Subroutine is a set of instructions in a computer program that performs a specific task and can be called from multiple points within the program. From sub- plus routine. Equivalent to a function or procedure in modern programming. Using subroutines promotes code reuse and modularity. Fundamental concept in programming; the term is older, common in assembly and early&nbsp;languages.
